# コーディング標準

## MUST

* 各言語の公式または定評あるベストプラクティス（GoであればEffective Go等）に準拠すること。
  * ベストプラクティスの体系が複数ある場合は一つのリポジトリおいて一つに絞ること
* 機密情報（APIキーなど）は環境変数やSSM Parameter Storeなどの外部サービスから実行時に取得すること。
  * 逆に言うと、.envに平文で書いて保存しないこと。
  * 暗号化して保存することは許容されるが、復号鍵をプロジェクトにバンドルしないこと。また、AIエージェントが閲覧可能な場所に復号した後のファイルを一次的であっても置かないこと。
* すべての関数は、少なくとも1つのユニットテストから実行されること。
* すべてのテストが通ること。
* 仕様に記載されたすべての詳細を実装すること。
